| Take       | Take              | Take         | 无              | Take               | Take                |

| UBITS      | 无                | UBITS        | 无              | 无                 | User Bits           |
| Track Name | Audio Track Names | TRK1、TRK2…… | iXML Track List | Track 1、Track 2…… | Channel Names       |
